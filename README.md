# EventIn

## Deskripsi Aplikasi

**EventIn** adalah aplikasi manajemen event berbasis web yang dirancang untuk membantu proses pengelolaan acara secara digital. Aplikasi ini memungkinkan pengguna untuk melihat informasi event, melakukan pendaftaran, mengakses materi atau informasi terkait acara, serta membantu administrator dalam mengelola data event dan peserta secara lebih efektif.

Dengan sistem terpusat, EventIn mempermudah proses administrasi acara sehingga lebih cepat, terorganisir, dan mudah diakses oleh seluruh pengguna.

## Cara Menjalankan Aplikasi

### 1. Clone Repository

```bash
git clone https://github.com/username/eventin.git
```

### 2. Masuk ke Folder Project

```bash
cd eventin
```

### 3. Konfigurasi Database

Buat database MySQL baru kemudian import file SQL yang disediakan pada folder database.

### 4. Jalankan Web Server

Gunakan Laragon, XAMPP, atau Apache untuk menjalankan aplikasi.

### 5. Akses Aplikasi

```text
http://localhost/eventin
```
